
# Keno-Bet

Keno-bet is a keno game made with c++

## Compilation

1. In terminal "cd" into the project folder, after that writes the command 

```bash
g++ -Wall -std=c++11 game.cpp KenoBet.cpp -o game
```
 and press Enter.
## Usage

2. run the comand
```
./game 'default-bet.txt'
```
